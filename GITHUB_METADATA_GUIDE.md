## How to Add via GitHub Web Interface

### Step 1: Go to Repository Main Page
1. Navigate to: https://github.com/AyushSharma297/SpectraStrike-Hub
2. Look at the right side of the page for the **"About"** section

### Step 2: Edit the About Section
1. Click the **⚙️ gear icon** next to "About" (top right of About box)
2. A dialog will open with editable fields

### Step 3: Add Description
1. In the **Description** field, paste:
   ```
   A futuristic, glassmorphic smart lighting hub for unified control of WLED, WiZ, OpenRGB, and Windows Dynamic Lighting with real-time screen sync. Features scene management, group control, color palette generator, schedules, and MCP integration.
   ```
